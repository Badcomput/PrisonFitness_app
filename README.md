# 囚徒健身

用古老的真理锻造身体。

一款基于《囚徒健身》六艺十式的自重训练应用，无需器械，随时随地锻炼。

## ✨ 功能

### 🏋️ 六艺训练
六种自重训练类型，每种10个难度递进的招式：

| 动作 | 入门 | 进阶 | 高阶 |
|------|------|------|------|
| 俯卧撑 | 墙壁俯卧撑 | 膝盖俯卧撑 | 单臂俯卧撑 |
| 深蹲 | 半深蹲 | 标准深蹲 | 单腿深蹲 |
| 引体向上 | 水平引体 | 屈腿引体 | 单臂引体 |
| 举腿 | 卧姿举腿 | 悬垂举腿 | 悬垂直举腿 |
| 桥 | 起始桥 | 完整桥 | 合桥 |
| 倒立撑 | 靠墙倒立 | 偏倒立 | 自由倒立 |

### ⏱️ 训练计时
- 自动倒计时组间休息
- 完成提示音
- 可跳过休息

### 📊 训练记录
- 近7天训练统计图表
- 完整历史记录
- 按日期分组查看

### 🔔 每日提醒
- 自定义提醒时间
- 养成训练习惯

### 🎯 进度解锁
- 达标后解锁下一式
- 循序渐进的训练计划

## 📖 使用指南

### 开始训练

1. 打开应用，进入**首页**
2. 选择一个**动作类型**（如俯卧撑）
3. 选择当前**解锁的招式**
4. 设置**目标组数和次数**
5. 点击**开始训练**

### 训练过程中

- **完成本组**：记录当前组的训练
- **跳过休息**：快速进入下一组
- **结束训练**：保存训练记录

### 查看记录

- 点击底部**记录**标签
- 顶部查看近7天训练图表
- 下方是历史训练记录

### 设置提醒

1. 点击底部**关于**标签
2. 开启**每日提醒**开关
3. 选择提醒时间
